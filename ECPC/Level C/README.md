## Topics Covered

You can solve these problems using one or more of these core topics (listed in no specific order):

1. Game Theory (Nim-Game / Impartial Games)
2. Dynamic Programming (DP)
3. Graphs
4. Disjoint Set Union (DSU)
5. Greedy
6. Segment Tree
7. Bit Manipulation
8. Combinatorics
9. Geometry
10. Binary Search / Ternary Search
11. Number Theory
12. Mo's Algorithm (Square Root Decomposition)
13. Fenwick Tree (Binary Indexed Tree)
