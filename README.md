# AIU-ICPC-Handbook

A collection of algorithms and templates for competitive programming.

---

# Roadmap

## Level 1

* Prefix Sum & Frequency Array
* Standard Template Library (STL)
* Binary Search & Two Pointers
* Bitmask, Bitset & Bits
* Number Theory
* Greedy

## Level 2

* Recursion & Backtracking
* Basic DP & Advanced DP
* Graph Representation (DFS & BFS)
* Shortest Path
* Disjoint Set Union (DSU) & Minimum Spanning Tree (MST)
* Segment Tree & Sparse Table

---

# Prerequisites

* Basic programming knowledge in any language (e.g., C++, Java, Python, etc.)
