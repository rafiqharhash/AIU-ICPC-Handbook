# AIU-ICPC-Handbook
collection of Algorithms and template for Competitive programming.

# Roadmap

## Level 1

* Prefix Sum & Frequency Array
* Standard Template Library (STL)
* Binary Search & Two Pointers
* Bitmask, Bitset & Bits
* Number Theory
* Greedy

## Level 2

* Recursion & Backtracking
* Basic DP & Advanced DP
* Graph Representation, DFS & BFS
* Shortest Path
* DSU & MST
* Segment Tree & Sparse Table

# prerequisite : 

Just the basic of programming with any language (C++, Java, Python , eg..)
